Данный проект под названием todolist является веб-приложением и выполняет функции планировщика задач.
В нем реализован весь функционал согласно ТЗ, в т.ч. задачи со звездочкой (если я ничего не пропустил).

Для сборки образа todolist необходимо и достаточно выполнить следующую команду docker из корневой дирректории проекта: "docker build -t todolist .".
Для запуска контейнера из собранного на предыдущем шаге образа todolist необходимо и достаточно выполнить следующую команду docker из корневой дирректории проекта: "docker run --env-file .env -p 7540:7540 todolist".

Адрес начальной страницы приложения: "http://localhost:7540/login.html".
В файле .env можно переопределить значения некоторых переменных окружения.
